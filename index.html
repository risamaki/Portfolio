Meetup App

An Android Application that helps students meet up with their peers on campus. It compares both students schedules to find a time and place (cafe or resturant) in a certain radius where they could both meet up. We utilized the MapQuest API to find all the possible locations to meetup and the FourSquare API to incorprate information about the meetup spot such as name,address, ratings and reviews.

Skills & Software used: Java, Android Studio, JSON and XML Parsing, API Integration

Testing: JUnit
